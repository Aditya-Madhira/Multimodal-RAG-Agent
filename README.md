# Multimodal RAG Agent

A powerful, end-to-end e-commerce shopping agent leveraging **Multimodal AI** (vision, audio, and text). This application allows users to search for products using their voice, text queries, or image uploads, utilizing a **Retrieval-Augmented Generation (RAG)** architecture and advanced vector search capabilities.

## 🚀 Features
- **Multimodal Search:** Query the product catalog using text, speech (audio transcription), or images.
- **RAG Architecture:** Employs Retrieval-Augmented Generation to fetch context-aware results.
- **AI Agent:** A backend agent dynamically routes queries and interacts with data.
- **Vector Search:** Fast and precise product retrieval through embeddings.
- **Interactive UI:** A highly responsive frontend interface built for a seamless user experience.

## 🛠️ Tech Stack
- **Backend:** Python, LangChain/LlamaIndex, Vector Database, Vision and Audio Embeddings
- **Frontend:** Streamlit 
- **AI / ML:** Multimodal LLMs, Speech-to-Text Transcription

## 📂 Project Structure
- `/backend` - Core logic including the AI agent, RAG retrievers, and embedding services.
- `/frontend` - Interactive Streamlit application.

## 💡 Use Case
Perfect for modern e-commerce or retail applications where customers need flexible, intelligent, and highly accurate ways to search for products beyond traditional keyword text search.
